## Usage

```bash
# Tap this repository (one-time)
brew tap ImKKingshuk/tap

# Install a formula
brew install lockknife

# Or install directly without tapping
brew install ImKKingshuk/tap/lockknife

# Update
brew update && brew upgrade lockknife
```

## Requirements

- macOS (Apple Silicon)
- Homebrew 4.0+
